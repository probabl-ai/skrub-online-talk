## Getting Started

### Install pixi

First, install pixi if you haven't already:

```bash
curl -fsSL https://pixi.sh/install.sh | bash
```

### Start JupyterLab

To start JupyterLab and work with the notebooks:

```bash
pixi run jupyter lab
```

Notebooks are located in the `notebooks` directory.
